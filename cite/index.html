Atlas / Cite

Citation guidance

Cite the exact view.

Each atlas item is designed to have a stable URL. Until the manuscript metadata and public release version are finalized, use the page title, atlas title, URL, and access date.

In the manuscriptLink to the narrowest relevant page—the exact figure, table, type profile, or comparison—not only the atlas home page.
For reproducibilityRecord the release or commit identifier when a view is used for a submitted manuscript version.
For downloadable materialKeep the page URL with exported PNG, SVG, CSV, or JSON files so readers can recover their context.

A formatted IEEE citation, authorship, DOI or archival identifier, and release history will be added when those details are confirmed.
