Version 1.0.0

21 July 2026

Initial public PINN Review Atlas foundation.

Identity preservation

The review’s numeric reference ID is the stable primary key. Sorting and filtering never renumber records.

Link verification

External destinations use HTTPS DOI resolvers or authoritative publisher records. Missing or uncertain destinations are not guessed.

Access classification

Open-access and subscription labels reflect the publisher or source record at the time of verification. Ambiguous cases remain “Not verified”.

Citation formatting

The reader-facing register follows the MDPI ACS reference structure while preserving the bibliographic facts available in the source document.

Corrections

Corrections should identify the reference ID, affected field, proposed value, and an authoritative DOI or publisher source.

Reproducibility

The full JSON and machine-readable metadata are downloadable. Client-side exports are generated directly from that same dataset.
